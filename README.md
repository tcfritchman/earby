# Earby. #

* A web application to aid musicians in learning music by ear.

### Features ###

* Simple playback
* Play/pause/skipforward/backward
* Static waveform view
* 1 region with markers
* Set start/end
* Goto start/end
* Loop region
* Load files from disk
